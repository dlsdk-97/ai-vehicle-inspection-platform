# AI Vehicle Inspection Platform

KT AIVLE School Big Project

## 프로젝트 소개
완성차 생산 라인에서 발생하는 불량을 AI로 자동 검수하고
관리자가 결과를 확인할 수 있는 플랫폼 개발

## Tech Stack

- React
- Python
- MySQL
